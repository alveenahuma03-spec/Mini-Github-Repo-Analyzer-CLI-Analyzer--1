GitHub Repo Language Analyzer

A simple Python script that takes a GitHub repository URL and shows the programming languages used in that repo.

Features
- Extracts repository owner and name from a URL
- Uses GitHub API to fetch language data
- Displays all languages used in the repository

Requirements
- Python 3
- requests library

Install dependencies:

pip install requests

Run the script:
python analyzer.py

Enter a GitHub repository URL when prompted:
Enter GitHub repo URL: https://github.com/user/repo
